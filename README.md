# Python Projects Collection 🐍💻

This repository contains a collection of Python projects that I built while learning and practicing programming. Each folder contains a standalone project demonstrating a specific concept, library, or idea.

---

## 📁 List of Projects

1. **Birthday Wisher**  
   Sends automated birthday emails using Python's datetime and SMTP libraries.

2. **Kanye West Quote Generator**  
   Displays random Kanye West quotes using an API and tkinter GUI.

3. **Mile to Kilometer Converter**  
   A simple GUI app that converts miles to kilometers using tkinter.

4. **Password Manager**  
   A secure password manager with file storage and password generation.

5. **Pomodoro Timer**  
   A productivity timer using the Pomodoro technique with a tkinter interface.

6. **Pong Game**  
   Classic Pong game built using Python's turtle graphics.

7. **Trivia App**  
   A quiz application that fetches trivia questions from an API.

8. **Turtle Crossing Game**  
   Frogger-style crossing game using turtle module.

9. **Turtle Race Game**  
   A simple turtle race simulation with random movement.

10. **U.S. States Game**  
   Interactive game to guess all U.S. states using the turtle graphics module.

11. **Website Template**  
   A basic front-end website layout using HTML and CSS (non-Python).

12. **Snake Game**  
   Classic snake game implemented with the turtle module.

---

## 📌 Notes

- All projects are built with beginner to intermediate-level Python.
- Most of them use libraries like `tkinter`, `turtle`, `smtplib`, and `requests`.
- Perfect for learning logic, OOP, API integration, and GUI design.
